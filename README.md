# Emailer

Deploy your own **Free email server** on Cloudflare. Support Worker Free and Paid Plan:

| Deploy                 | Feature | Cost  |
| ---------------------- | ------- | ----- |
| Cloudflare Worker Free | ✅Full  | 0     |
| Cloudflare Worker Paid | ✅Full  | 5$/mo |

When you choose deploy via Worker Free, you need to get your [Resend API Token](https://resend.com/emails), because cloudflare didn't provide email sending to free accounts.

And keep in mind, Resend has 3000 of sending limit per day, which is okay if it's for your own use.

## Techs

- Tanstack Start (React)
- TailwindCSS
- Better Auth
- Shadcn (BaseUI)
